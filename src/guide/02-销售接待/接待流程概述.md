# 房地产销售接待基本流程概述

## 第一节 迎接客户

### 一、基本动作

1. 主动招呼"欢迎参观"，提醒其他销售人员。
2. 当值人员立即上前，热情接待。
3. 协助客户放置雨具、衣帽等物品。
4. 初步判断客户意向，了解来访渠道。

### 二、注意事项

1. 仪表端正，态度亲切。
2. 接待人数不超过 2 人（一主一副）。
3. 所有来访者均应提供资料，保持热情。
4. 随时保持现场整洁和良好仪表。

## 第二节 介绍产品

### 一、基本动作

1. 交换名片，了解客户基本信息。
2. 按规划路线配合销售道具介绍产品，重点说明地段、环境、交通、配套、楼盘功能、建材等。

### 二、注意事项

1. 强调楼盘整体优势。
2. 用热忱与诚恳建立信任关系。
3. 把握客户真实需求，快速制定应对策略。
4. 多人时注意区分决策者，把握相互关系。

::: info
[点击查看项目讲解标准化细则](/guide/02-销售接待/项目讲解)
:::

## 第三节 带看现场

### 一、基本动作

1. 结合工地现况和周边特征边走边介绍。
2. 按户型图让客户切实感受所选户型。
3. 持续讲解，保持客户注意力。

### 二、注意事项

1. 事先规划带看路线，注意整洁与安全。
2. 嘱咐客户戴好安全帽，带好随身物品。

::: info
[点击查看带看现场标准化细则](/guide/02-销售接待/带看现场)
:::

## 第四节 购买洽谈

### 一、基本动作

1. 倒茶寒暄，引导客户入座。
2. 主动选择一户作试探性介绍。
3. 根据客户喜好作详尽说明。
4. 针对疑惑点解释，克服购买障碍。
5. 适时营造购买气氛，强化购买欲望。
6. 在客户认可度达 70% 时，说服其下定。

### 二、注意事项

1. 安置客户在视野愉悦、便于控制的位置。
2. 销售资料和工具准备齐全。
3. 了解客户真实需求和主要问题点。
4. 与同事配合，让经理知晓客户意向房号。
5. 判断客户诚意、购买力和成交概率。
6. 气氛营造自然亲切，掌握火候。
7. 不夸大、不虚构。
8. 职权外承诺需报经理批准。

::: info
[点击查看购买洽谈标准化细则](/guide/02-销售接待/谈判逼定)
:::

## 第五节 暂未成交

### 一、基本动作

1. 提供完整资料供客户考虑。
2. 告知联系方式，承诺提供购房咨询。
3. 对有意客户约定下次看房时间。
4. 送客至大门外。

### 二、注意事项

1. 对所有客户保持亲切态度，始终如一。
2. 及时分析未成交原因并记录。
3. 向经理报告，采取补救措施。

::: info
[点击查看送客出门标准化细则](/guide/02-销售接待/送客出门)
:::

## 第六节 填写客户资料表

### 一、基本动作

1. 每接待完一组客户后立即填写资料表。
2. 填写重点：
   - 客户联络方式和个人信息
   - 客户对楼盘的要求条件
   - 成交或未成交的真正原因
3. 将客户分类为 A（很有希望）、B（有希望）、C（一般）、D（希望渺茫）四个等级。
4. 一联交经理备案，一联自己留存。

### 二、注意事项

1. 资料表应详尽填写，妥善保存。
2. 客户等级应阶段性调整。
3. 定期召开会议，检查销售情况，采取应对措施。

::: info
[点击查看接待总结标准化细则](/guide/02-销售接待/接待总结)
:::

## 第七节 客户追踪

### 一、基本动作

1. 按客户等级联系，向经理口头报告。
2. A、B 级客户列为重点，保持密切联系，努力说服。
3. 详细记录追踪情况，便于分析判断。
4. 婉转要求客户帮忙介绍。

### 二、注意事项

1. 注意话题选择，避免给客户造成死硬推销印象。
2. 追踪间隔一般 2-3 天为宜。
3. 变化追踪方式：电话、寄资料、上门拜访、邀请参加活动等。
4. 多人联系同一客户时应相互通气，协调行动。

::: info
[点击查看客户追踪标准化细则](/guide/02-销售接待/跟进维护)
:::

## 第八节 成交收定

### 一、基本动作

1. 客户决定购买时，通过销控告知经理。
2. 恭喜客户。
3. 收取小定金或大定金，告知双方行为约束。
4. 详尽解释定单各项条款：总价款、定金金额、补足日期、签约日期、折扣、付款方式等。
5. 客户、销售人员、经理三方签名确认。
6. 将定单连同定金交经理点收备案。
7. 将客户联交客户，告知补足或签约时带来。
8. 确定补足日或签约日，告知注意事项和所需证件。
9. 再次恭喜客户。
10. 送客至大门外。

### 二、注意事项

1. 与经理和同事配合，营造现场气氛。
2. 定单一式四联：客户联、公司联、工地联、财务联。
3. 客户未带足金额时，可收小定金（300-5000 元），保留期 3 天。
4. 大定金下限 1 万元，保留期 7 天，金额越多越好。
5. 定金为合约一部分，任一方违约需赔偿。
6. 定金与签约间隔尽可能短，防止节外生枝。
7. 折扣或附加条件需报经理备案。
8. 填写完后仔细检查户别、面积、总价、定金等。

## 第九节 定金补足

### 一、基本动作

1. 定金栏填写实收补足金额。
2. 划掉定金补足日及应补金额栏。
3. 确定签约日期和金额，填写于定单。
4. 若重开定单，依据小定金定单内容填写。
5. 告知签约注意事项和所需证件。
6. 恭喜客户，送至大门处。

### 二、注意事项

1. 补足日前再次联系确认。
2. 检查户别、面积、总价、定金等。
3. 向经理汇报备案。

## 第十节 换房与退房

### 换房

#### 一、基本动作

1. 告知换房需公司审批，原则上不允许。
2. 若客户坚持，确认符合"以小换大"或"以低换高"原则。
3. 填写换房申请单，注明对比信息（户型、面积、总价）。
4. 报经理及上级审批。
5. 审批通过后，填写换房后的户型、面积、总价。
6. 重新计算应补金额及签约金。
7. 注明原房号换至新房号。

#### 二、注意事项

1. 严格审批流程，不得擅自承诺。
2. 检查户别、面积、总价、定金、签约日等。
3. 收回原定单作废。

### 退房

#### 一、基本动作

1. 严正表明立场：原则上不同意退房，除非开发商违约。
2. 告知"定金罚则"：客户原因退房，定金不退。
3. 深入分析退房原因，尝试挽回客户。
4. 确实无法挽回且符合条件时，报经理或上级确认。
5. 办理退房手续，结清款项。
6. 收回作废合同，交公司备案。
7. 保持职业素养，送客至大门外。

#### 二、注意事项

1. 强调退房不退定金原则，除非法律规定的特殊情况。
2. 资金退还或转移需双方签名确认。
3. 有争议时可申请仲裁或法院裁决。

## 第十一节 签订合约

### 一、基本动作

1. 恭喜客户。
2. 验证身份证原件，审核购房资格。
3. 出示预售合同文本，逐条解释条款：当事人信息、房产位置面积、土地性质、使用期限、规划用途、房屋布局结构质量、装饰标准、配套设施、价格支付方式、交付日期、违约责任、争议解决方式等。
4. 与客户商讨确定内容，在职权范围内适当让步。
5. 签约成交，收取首期房款，抵扣已付定金。
6. 收回定单，交经理备案。
7. 协助办理登记备案和银行贷款。
8. 备案且办好贷款后，交客户一份合同。
9. 恭喜客户，送至大门外。

### 二、注意事项

1. 事先准备好示范合同文本。
2. 事先分析可能问题，向经理报告并研究解决办法。
3. 客户问题无法说服时，汇报经理或上级。
4. 最好由购房户本人填写并签名盖章。
5. 他人代理签约需公证委托书。
6. 解释条款时侧重客户立场，增强认同感。
7. 签约后迅速交房管机构审核并登记备案。
8. 牢记：登记备案后买卖才正式成交。
9. 签约后保持联系，帮助解决问题并让其介绍客户。
10. 问题无法解决时，让客户先回，另约时间，以时间换取折让。
11. 及时检查签约情况，采取应对措施。

## 第十二节 提供超值服务

超值服务可包括以下方式：

1. 站在客户立场提供咨询服务。
2. 提供客户所需信息。
3. 逢年过节寄卡片、赠送小礼品。
4. 主动寻求信息反馈并提供所需服务。
5. 替客户做延伸服务，使其体会"超值"。
6. 在业务和道德允许范围内，提供办理私人事务的方便。

## 第十三节 交房入伙

### 一、基本动作

1. 提前通知交房时间、地点及所需证件资料。
2. 核对客户身份及合同，确认房款已结清。
3. 陪同验收房屋：检查质量、设施设备、装修标准等是否符合合同。
4. 协助办理物业交接，签署《房屋交接书》。
5. 移交钥匙、使用说明书、质量保证书等资料。
6. 介绍物业公司及服务内容，协助办理物业手续。
7. 告知水、电、气等使用方法及缴费方式。
8. 解答入住疑问。
9. 恭喜客户乔迁新居。

### 二、注意事项

1. 交房前确保房屋达到交付标准，设施设备运行正常。
2. 验房时认真仔细，发现问题及时记录并承诺整改时间。
3. 对客户合理要求积极协调解决。
4. 交接手续完整规范，文件资料齐全。
5. 保持良好关系，为客户转介绍打下基础。
6. 及时反馈交房情况给相关部门。
7. 质量问题及时跟进处理，确保客户满意。
