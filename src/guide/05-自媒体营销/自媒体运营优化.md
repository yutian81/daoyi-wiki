# 朋友圈、抖音运营优化措施

## 一、朋友圈运营优化措施

### 第一，头像和名称很重要

- 1. 头像
用自己的正装真实头像，这样比一张风景照来得真实可信，然后头像背景照片，可以选一张楼盘或者售楼处的门面照、LOGO照。

- 2. 昵称
直接写上：A王先生【房产顾问】，有利于传递真实、靠谱的印象。

- 3. 个人简介
写上自己的介绍：比如房产置业、投资、联系电话xxxxx ；联系电话这个非常重要，有这样几种放置的方法：
**方法一**：可以放在头像照片的最下方；
**方法二**：可以放在昵称的右上角【标成类似平方数的小字】；
**方法三**：直接放在朋友圈显示的地理位置中，位置信息那里可以直接拨打电话，很方便；
**方法四**：写在个人简介中。
**注意**：不必在每条房产信息里面加电话，有限的文字需要用来描述具体的楼盘信息。

### 第二，对好友进行分组

重要的一条原则：分组发布。发给谁看比多少人看更有价值。
也没必要将房子的信息一条又一条的推到老家亲戚、朋友、家人的朋友圈中，这样做也没有任何意义。
**【分组建议】**：客户分一组，朋友同事分一组，家人分一组，其他人分一组。
**【注意】**：客户要细分，甚至可以细分为：家住哪片区域的客户，哪个年龄段的客户，什么需求的客户，或者哪个时间段来看房子的客户等等。每一位客户都要备注好，尤其是主动联系你的客户，从未谋面的客户，一定要备注好时间信息，让自己能够回忆起来。

### 第三，不要刷屏，那发什么

不能天天在朋友圈打广告，频繁的刷屏会让人讨厌，这也是容易被人屏蔽的最直接原因。发朋友圈的原则：
- A.在朋友圈树立一个良好的正面形象；
- B.有目的性的撒网，从而找到、捞到买房客户；
- C.让客户想买房子的时候，第一时间找到我们。

那么，我们每天要发一些什么？发有价值的。

- 1、多发能够帮助到“粉丝”的东西
**【知识类】**
比如分享买房子要注意的事项，禁忌。可以自己根据行业内的经验写一小段“看房时应该注意什么？”，让客户能够学到几招。
比如可以分享行业内的咨询，政府对房产的调控，省市区的行业内重大新闻等等。
也可以分享户型知识，物业知识，交房知识等等。
总结：经验类的，资讯类的，房产专业知识类的，都很适合发朋友圈。
**【故事类】**
比如今天遇到一个什么特殊客户，什么需求，最后是如何解决的，写的有故事性一些，能够吸引大家看。
大家都有一个从众的心理。自己不买，看看别人是怎么买的，自己长长知识和经验是肯定不排斥的，如果故事写的真实可信，客户会更加信服【发这个故事的人】，好感度一下子就上来了。
**【心灵鸡汤类】**
心灵鸡汤要适当的发。没有鸡汤也不现实，朋友圈有一点鸡汤最好，别太多。心灵鸡汤可以作为朋友圈的点缀。

- 2、发轻松、有趣的内容
每天的压力这么大，生活节奏这么快，整个人已经神经崩的很紧张了，如果想放松一下，看看朋友圈，就想看到一些有趣的、轻松的，让人开怀大笑的东西。

- 3、发真实的自己
我们屏蔽一些人的时候，理由一般是什么？多半都是垃圾朋友圈太多，朋友圈全是广告，一点儿自己的信息都没有，一点儿也不真实。朋友圈发就发真实的自己，越真实越让人觉得可信服。
**内容一：真实本分**
可以写写自己的工作，可以拍一下自己加班的场景，可以发同事聚餐，也可以发自己的踌躇满志，发工资庆祝等等。每个看到你朋友圈的客户都会判断你是否诚实， 如果能让他们能看到你的成长，信服度会增加很多。
现在朋友圈有了一个【显示朋友圈】的选项，可以选择【显示最近三天的】【显示最近一个月的】【显示最近半年的】【显示最近一年的】。
*【建议】*：选择【显示最近一年的】，因为时间越长，客户了解你的信息就越全，也会越放心。

**内容二：朋友圈评论沟通感情**
不要让大家觉得你成天都是在卖卖卖，卖房子。要多和朋友圈的客户沟通感情，有这样几个小技巧：
*【点赞】*：多给朋友圈的客户点赞，让客户能够时时刻刻看到你，提高自己的曝光度。
*【评论】*：有空的时候在客户的评论里拉拉自己的家常，客户看到的几率会大大增加，是增进感情的机会。
*【回复评论】*：客户如果给你评论了，这是很好的沟通机会，一定要回复评论，这样一来一回，能够经营彼此之间的信任和感情 。但是，切忌一上来就推销自己的房子。

### 第四，发布楼盘信息

像卖面膜一样发一段话，然后配上好几张图片，这样的楼盘信息，在朋友圈里已经太落后了，而且如果图片很大，大家点开看还要缓冲，很麻烦。还有很关键的一点，这样发的话，特别难转发。
那该怎么发布最重要的楼盘信息、促销信息呢？

- 1. 标题
每个人都是标题党，这个毋庸置疑。所以，发每一条朋友圈的时候，标题是最最关键的。我们设计出来一些点击率高的标题，然后每个人循环套用就会比较高效。例如：
特惠！特惠！今天截止……
你从来不知道的那些【买房子的坑】
开盘之前，开发商绝对不会告诉你的三件事……
最后一套！122平三房，首付约XX万

- 2. 楼盘描述
楼盘信息发送注意：
楼盘介绍需要换行显示，不要连着写，这样看起来不够清晰、直观；
楼盘配图尽量清晰和详尽（但是图片文件不能很大），好看的图越多越好；
楼盘图片上面可以适当配一些文字介绍；
定向推广，让每条朋友圈更有价值。我们对好友进行了分组，如果我们发布了房子信息， 可以只有【客户】这个分类能看到。

- 3. 提醒
发送朋友圈的最下方还有一个@提醒谁看功能，这个功能可以提醒10个好友查看此动态，我们销售可以找十个朋友圈的意向客户，设置提醒功能。

- 4. 价格
发朋友圈严禁报具体房子的价格，但是有些客户私聊我们会一直追问价格，这个时候可以运用一些【逼访】的小技巧。例如：
“咱们俩聊个微信，肯定不可能把房子定下吧？所以房子关键是要来看，什么时候有时间？星期六下午吧？”
“本身咱们就是一房一价，具体的到现场来看了，相中哪一套给你看哪一套的价格……”
“现在这个行情下，无论买得起买不起，都得先过来看看，如果想自己能买的起才来看房子，那根据我的经验，肯定很被动，好房子很容易就被别人买走了，相信我，您先过来看看。”

- 5. 图片
图片分为两种：
一种是设计师设计好的图片，大家统一发；
另一种比较真实，也更加适合发朋友圈：自己随手拍的楼盘实景。

## 二、抖音运营优化措施

### 第一，抖音账号包装

- 1. 头像
真人头像更权威、职业化，或者直接用企业头像、品牌logo等。

- 2. 昵称
账号名不要太长，尽可能的其中包含行业关键词，以便提高实时搜索排名，展现形式多为：真名、名字+行业关键词、名字+地区+行业关键词、名字+技能。

- 3. 个性签名
这直接展示你是做什么的，也是增强账号人设形象的窗口，所以要价值最大化的介绍账号和站外引流。

- 4. 背景
选用与整体打造账号有所关联的图片，设置引导咨询、引导关注、增加权威、强调价值，但切记要与账号内容的调性一致、与行业相关，合适的背景会对账号发布检索时排名效果有显著提升。

### 第二，5种爆款视频类型

- 1. 房产知识讲解类
此类视频最常见的就是分享房地产经济类的干货知识、避坑宝典，还有房地产息息相关的政策分享及见解等。

- 2. 讲故事类
买卖房产发生的趣事、大无语事件等，通过分享这些事从侧面带出买卖房屋的实用常识。

- 3. vlog探店类
通过实地探房进行专业的讲解，对抢手楼盘进行测评，并发布一些买房建议。

- 4. 热点事件类
紧跟热点事件，包括但不限于最新的楼市资讯，地产要闻等，站在购房者的视点剖析事件，发表自己的观点与建议。

- 5. 创意剧场类
把买卖房子的相关事件或资讯拍成创意小短剧。
其实总结起来爆款视频的制作逻辑无非就3点：①有用 ②有趣 ③真实。
而我们要持续性输出优质视频，就得要求他成本低、制作简单且易变现，其中知识讲解类和vlog探店类视频就是最符合这个要求的，同时也符合爆款逻辑中的2点：有用、真实。
这类视频的表现形式就2种：①真人出镜口播，②房屋展示素材+文案。

### 第三，提升播放率和线下转化率

- 1. 搭配热门音乐，加热门标签，@官方助手，选择黄金时间发布视频

- 2. 蹭热点

- 3. 视频标题很重要
我们看视频往往会被一个好的标题吸引，甚至会因此把视频看完，提高完播率，尤其是能引起共鸣，引发好奇，巨大反差，抓痛点等标题，建立选题库，爆款视频肯定不是打造一个就够了，所以建立选题库是未雨绸缪。
比如：
**① 引发共鸣**：小白买房最关心但最想不明白的5个问题
**② 引发好奇**：和杨幂做邻居要多少钱？
**③ 巨大反差**：你根本买不到70年产权的房子
**④ 抓痛点**：房价20年涨了20倍，今年还能买房子吗？看清房价未来趋势

- 4. 投“DOU+”，注意DOU+是锦上添花不是雪中送炭

- 5. 评论区回复引导
不仅要在自己的作品下及时回复，还要去同行的作品下，或者同类型视频的评论区中回复引导有意向的用户，这样能大大提高自己的转化率和私域流量。
